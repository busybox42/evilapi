Welcome to Evil Admin Tools

A comprehensive email system administration toolkit designed for email infrastructure research, troubleshooting, and analysis. Built for email administrators, system administrators, and IT professionals managing email systems.

⚠️ Important Notes:

  • EvilAPI enforces rate limits to ensure system stability
  • All tools are intended for legitimate email system administration and research
  • Use responsibly and only on email systems you own or have permission to administer

Select a tool from the navigation bar above to get started with your email system administration tasks.

API Documentation

      
This page will contain detailed documentation for the EvilAPI endpoints.

      
      
Available Endpoints:

      
    
          
  • /api/email/info: Get email configuration information.
    • 
          
  • /api/smtp/test: Perform SMTP tests.
    • 
          
  • /api/blacklist/check: Check email blacklists.
    • 
          
  • /api/header/analyze: Analyze email headers.
    • 
          
  • /api/spam/scan: Scan email for spam.
    • 
          
  • /api/dmarc/validate: Validate DMARC records.
    • 
          
  • /api/dkim/validate: Validate DKIM records.
    • 
          
  • /api/dkim/generate: Generate DKIM keys.
    • 
          
  • /api/email/delivery: Test email delivery.
    • 
          
  • /api/auth/validate: Validate authentication.
    • 
          
  • /api/whoami: Get client IP and network info.
    • 
          
  • /api/dns/lookup: Perform DNS lookups.
    • 
          
  • /api/dns/propagation: Check DNS propagation.
    • 
          
  • /api/ssl/validate: Validate SSL certificates.
    • 
          
  • /api/pgp/encrypt: Encrypt PGP messages.
    • 
          
  • /api/pgp/decrypt: Decrypt PGP messages.
    • 
          
  • /api/pgp/generate-temp-key: Generate temporary PGP keys.
    • 
          
  • /api/hash/create: Create password hashes.
    • 
          
  • /api/hash/validate: Validate password hashes.
    • 
          
  • /api/portscan: Perform port scans.
    • 
          
  • /api/network/ping: Perform network ping.
    • 
          
  • /api/network/traceroute: Perform network traceroute.
    • 
          
  • /api/whitespace/remove: Remove whitespace from text.
    • 
          
  • /api/base64/encode: Encode Base64.
    • 
          
  • /api/base64/decode: Decode Base64.
    • 
          
  • /api/url/encode: Encode URLs.
    • 
          
  • /api/url/decode: Decode URLs.
    • 
          
  • /api/time/convert/epoch-to-date: Convert epoch to date.
    • 
          
  • /api/time/convert/date-to-epoch: Convert date to epoch.
    • 
          
  • /api/ssl/scan: Scan SSL/TLS vulnerabilities.
    • 
          
  • /api/speedtest/download: Perform download speed test.
    • 
          
  • /api/speedtest/upload: Perform upload speed test.
    • 
      


      
For more detailed information on each endpoint, including request/response formats and examples, please refer to the docs/API_Documentation.md file in the project repository.
